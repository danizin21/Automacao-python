# Automação com Python

Projeto de automação desenvolvido em Python com o objetivo de automatizar tarefas repetitivas, aumentar a produtividade e otimizar processos manuais.

# Funcionalidades

* Automação de tarefas utilizando Python
* Interação automática com teclado e mouse
* Manipulação de dados e planilhas
* Execução automatizada de processos repetitivos
* Integração com bibliotecas de automação

# Tecnologias Utilizadas

* Python
* PyAutoGUI
* Pandas

# Objetivo do Projeto

Este projeto foi desenvolvido com foco em aprendizado prático de automação utilizando Python, aplicando conceitos de produtividade, manipulação de dados e automação de processos.

# Como Executar

1. Clone este repositório:
git clone https://github.com/seuusuario/Automacao-python.git

2. Instale as dependências:
pip install pyautogui pandas

3. Execute o projeto:
python main.py
